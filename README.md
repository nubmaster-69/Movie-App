*A simple MVVM Android app using Kotlin*

## Main Features
- Get Today or This Week trending videos, popular using [TMDB API](https://www.themoviedb.org/documentation/api)
- View details of a movie

## Technologies Used:
- Kotlin
- Retrofit
- View model + Live Data
- Coroutine
- Endless Scroll
